Menentukan Harga Satuan Produk dengan SPLTV

Misal:
X = Harga satu lampu hias
Y = Harga satu mug ecoprint
Z = Harga satu kalender

Model SPLTV:
2x + y + z = 221.500 … (1)
x + 3y + 2z = 293.000 … (2)
3x + 2y + z = 311.500 … (3)

Menyelesaikan (1) dan (3)
2x + y + z = 221.500
3x + 2y + z = 311.500
--------------------------- —
-x - y = -90.000
90.000 = x + y … (4)

Menyelesaikan (1) dan (2)
2x + y + z = 221.500 |×2| 4x + 2y + 2z = 443.000
x + 3y + 2z = 293.000|×1| x + 3y + 2z = 293.000
--------------------------- –
3x - y = 150.000 … (5)

Substitusi Y ke Pers. (5)
3x - (90.000 - x) = 150.000
3x - 90.000 = x = 150.000 4x = 150.000 + 90.000 4x = 240.000 x = 240.000 : 4 x = 60.000

Substitusi X ke Pers. (4)
90.000 - 60.000 = y 90.000 - 60.000 = y 30.000 = y

Substitusi Z ke Pers. (1)
2x + y + z = 221.500
2 (60.000) + 30.000 + z = 221.500 120.000 - 30.000 + z = 221.500 z = 221.500 - 120.000 - 130.000 z = 71.500

Jadi, harga satu lampu hias adalah Rp 60.000

Jadi, harga satu mug ecoprint adalah Rp 30.000

Jadi, harga satu kalender adalah Rp 71.500
